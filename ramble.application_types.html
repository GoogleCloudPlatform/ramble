build_used_variables(workspace)[source]

Build a set of all used variables

By expanding all necessary portions of this experiment (required / reserved keywords, templates, commands, etc…), determine which variables are used throughout the experiment definition.

Variables can have list definitions. These are iterated over to ensure variables referenced by any of them are tracked properly.

Parameters:

workspace (Workspace) – Workspace to extract templates from

Returns:

All variable names used by this experiment.

Return type:

(set)
